# App móvil (Android + iOS)

Esta carpeta contiene una app base de Expo/React Native para consumir el backend CRM.

## Ejecutar

1. Instalar Expo CLI y dependencias de React Native.
2. Copiar `App.js` en un proyecto Expo (`npx create-expo-app`).
3. Ejecutar `npx expo start` y abrir en Android o iOS.

> Ajusta `API_URL` en `App.js` para que apunte a la IP de tu servidor.
